# ToDo List

## To run this project:

- First, use a Virtual Environment:

  ```
  python -m venv venv
  ```

- Then, activate it:

  ```
  source venv/bin/activate
  ```

- Second, install the Python dependencies:

  ```
  pip install -r requirements.txt
  ```

- Finally, run the server:

  ```
  uvicorn main:app --reload
  ```
